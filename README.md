Trabalho: Aplicação web que consome a API catfact.ninja. O usuário informa um número (20–250) e recebe um fato sobre gatos com aquele limite de caracteres. Feita com HTML, CSS e JavaScript puro. 

API: catfact.ninja/fact?max_length={numero} → retorna { fact, length }
